# 最优化方法

### 课程描述

- 必修，32学时
- 成绩：平时小测验（30%）+ 结课大作业（70%）【无考试】
- 平时小测验（30%）
  - 每1~2周在乐学平台进行1次测验【有截止日期】
  - 每次4～6选择题，较简单，类似Coursera等网课平台
-  结课大作业（70%）
  - 代码成绩占20%：在测试用例上性能占10%、代码注释10%
  - 报告占50%：报告内容的规范性、逻辑清晰程度等
