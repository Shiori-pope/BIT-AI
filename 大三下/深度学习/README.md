### 课程描述

- 必修，一周3节，8个实验+对应报告，无考试（但愿）
- 每两周一个实验ddl
- 一个bonus加分实验100+10（组队完成）
- 基于pytorch框架
- 本着项目初衷，这里我仅放了一些实现的关键代码和参考资料，可以看看
- 用Vscode里的codesnap插件截代码，很好看
