# 小学期工程实训

### 课程描述

- 10天，包含3-4天课程讲解+6-7天做软件
- 小组合作，5~6人，
- 完成10天的个人日报、结合测试文本、项目展示PPT、需求跟踪矩阵、可持续发展报告、实训报告总结
- 项目内容好像是即时聊天和邮件发送轮着来
- 需要答辩+项目展示



### 建议：

- 需要抱大腿，但是千万不要所有大腿抱到一起，否则很容易互相不服气
- 合作精神非常重要
- 项目难度不小，尽早开始讨论分工，不要想着休息一天，时间很紧张
- 大不了，答辩混一混！
- 一定要有靠谱的队友！！！
